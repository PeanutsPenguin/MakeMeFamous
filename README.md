# MakeMeFamous

Game jam réalisé à ISART Digital Paris. Une semaine pour faire un jeu avec le thème "Renverser les stéréotypes". Notre groupe etait compose de 3 Game Designers, 3 artistes, 1 Game Designer/Programmer et 1 Programmer

## Le jeu 
MakeMeFamous est un clicher 2D où vous incarnez un groupe de métal féminin essayant de percer dans le monde cruel de la musique. Afin de devenir populaire, vous devez produire des albums, sortir des singles ou encore faire des tournées, mais attention, il ne faut pas vous épuiser.
Devenez le groupe de metal le plus populaire du monde ! 

## Technique 
Le jeu a ete realsie sur Unity en 2D. Il contient plusieurs feature comme par exemple la sauvergarde de donne et le farm de ressources Passif !
